# Learn & Train a Tiny GPT（从零训练小 GPT）

面向零基础学习者的 **PyTorch 迷你 GPT** 教学项目：从数据处理、预训练、问答微调（SFT）到文本生成，全程可读代码、可本地复现。

**完整教程请阅读：[project/README.md](project/README.md)**

## 快速开始

```bash
pip install -r project/requirements.txt
python project/src/run_all.py
```
